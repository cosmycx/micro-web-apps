### Micro Web Apps:
	git clone

  #### build 1st app:
  cd reactapp1
  yarn install
  yarn build

  #### build 2nd app:
  cd reactapp2
  yarn install 
  yarn build

  #### start docker-compose from homeapp:

	cd homeapp
	docker-compose up
	open localhost in browser
