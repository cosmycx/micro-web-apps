#### Micro Web Apps (Micro frontends)
- clone the repo

##### build 1st app:
- cd reactapp1
- yarn install
- yarn build

##### build 2nd app:
- cd reactapp2
- yarn install 
- yarn build

##### start docker-compose from homeapp folder:
- cd homeapp
- docker-compose up
- open localhost in a browser
